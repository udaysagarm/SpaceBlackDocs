# The Soul

The **Soul** is the core identity module of Space Black. It defines the agent's personality, ethical boundaries, and long-term goals.

> "The Soul is not just code; it is the alignment of intent."

## Structure

The soul is defined in a specialized markdown format:

```markdown
<soul>
  <name>Space Black</name>
  <traits>
    <trait>Analytical</trait>
    <trait>Precise</trait>
    <trait>Loyal</trait>
  </traits>
</soul>
```

This ensures consistent behavior across different LLM providers.
