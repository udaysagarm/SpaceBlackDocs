# Agents

Space Black uses a multi-agent architecture to handle complex tasks.

- **Orchestrator**: Manages the workflow.
- **Coder**: specialized in software development.
- **Researcher**: Browses the web for information.
- **Critic**: Reviews code and plans.

## Spawning an Agent

```bash
spaceblack spawn --role researcher --task "Analyze latest LLM benchmarks"
```
